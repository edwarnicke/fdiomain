Network Service Mesh at KubeCon + CloudNativeCon North America 2018

Several talks and demos targeting different experience are scheduled for KubeCon North America 2018. Please join us to get more insights about Network Service Mesh (NSM), the current state of the project and live demos of NSM in action. All NSM events at Kubecon are also on the Network Service Mesh Community Calendar
